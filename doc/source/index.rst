Introduction
--------------------------------------
This is a simple implementation of an adaptative umbrella sampling method for calculating
free energy landscapes from molecular dynamics simulations.

Installation
--------------------------------------

Be aware that these modules depend on several standard scientific python packages, notably numpy,
scipy, matplotlib and pickle.
